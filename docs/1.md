# Data Type 

### 데이터의 크기



| bit           | byte       |
| ------------- | ---------- |
| 8bit          | 1 byte     |
| 1024 byte     | 1 kilobyte |
| 1024 kilobyte | 1 megabyte |
| 1024 megabyte | 1 gigabyte |
| 1024 gigabyte | 1 terabyte |

### 정수형



| 타입  | 메모리 크기 | 범위                             |
| ----- | ----------- | -------------------------------- |
| byte  | 1byte       | -128 ~ 127                       |
| short | 2byte       | -32769 ~ 32767                   |
| int   | 4byte       | -2147483648 ~ 2147483647         |
| long  | 8byte       | -922337........ ~ 922337........ |
